GENAI REFLECTION

OVERVIEW

For this assessment, I extensively used Claude (by Anthropic) as my GenAI tool to help build this Quarto website. This is my reflection on the problems I encountered and thoughts on how GenAI will shape my future work.

CHOICE OF GENAI TOOL

WHY CLAUDE?

I chose Claude (via claude.ai) as my main GenAI assistant for several compelling reasons:

— Strong Coding Capabilities: Best-in-class coding abilities among available GenAI tools

— Long Context Window: Handles extended conversations with excellent memory retention

I also occasionally used ChatGPT and Gemini, but Claude was my primary development assistant.

HOW GENAI HELPED ME

HTML/CSS STYLING ISSUES

Challenge: HTML and CSS have been challenging for me in the past. Although not inherently difficult, getting the layout and styling right requires significant time for research and testing.

How Claude helped:

— Quickly transformed my ideas into working HTML/CSS snippets

— Enabled rapid iteration through simple modification requests

Example transformation:

<!-- This didn't work -->
<div style="text-align: center;">
  <img src="image.jpg" style="width: 250px;">
</div>

<!-- Claude suggested this -->
![](image.jpg){width=250px style="..."}

BENEFITS VS CHALLENGES

BENEFITS

FASTER PROBLEM-SOLVING

Instead of searching through documentation and Stack Overflow, Claude provided direct answers, saving significant development time and reducing context switching.

LEARNING WHILE DOING

Claude didn’t just provide code - it explained concepts and reasoning, turning problem-solving into a learning opportunity.

ERROR DEBUGGING

Error debugging was always challenging for me. Claude helped identify and fix errors quickly, explaining the root causes along the way.

CHALLENGES

NOT ALWAYS CORRECT ON FIRST TRY

Some issues, like HTML rendering problems, required multiple iterations. Claude’s first suggestion didn’t always work, necessitating several alternative approaches.

CONTEXT LIMITATIONS

In longer conversations, Claude occasionally forgot earlier context or provided contradictory information that conflicted with previous suggestions.

OVER-COMPLICATED SOLUTIONS

Sometimes Claude over-engineered simple problems. For example, requesting a simple layout change resulted in an unnecessarily complex CSS grid solution.

FUTURE APPLICATIONS

FOR ACADEMIC WORK

Course Projects:

— Quick prototyping of ideas before deep implementation

— Debugging complex code errors efficiently

— Understanding new libraries or frameworks faster

— Getting explanations of academic concepts in simpler terms

Research:

— Literature review assistance (summarizing papers)

— Data analysis code generation

— Writing assistance for reports and documentation

FOR PROFESSIONAL DEVELOPMENT

Skill Building:

— Learning new programming languages quickly

— Understanding design patterns and best practices

— Exploring unfamiliar technologies and frameworks

Project Work:

— Accelerating prototype development

— Code review and optimization suggestions

— Documentation generation

FINAL REFLECTION

GenAI tools like Claude have fundamentally changed my development workflow. While they’re not perfect, the productivity gains and learning acceleration make them invaluable companions in software engineering education and practice. The key is understanding their limitations and using them as intelligent assistants rather than infallible oracles.
